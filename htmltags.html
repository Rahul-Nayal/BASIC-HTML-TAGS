List of HTML Tags

HTML Tag Description
<div> A container element used to group other elements.
<p> Defines a paragraph of text.
<h1> to <h6> Defines headers, with <h1> being the largest and <h6> the smallest.
<a> Defines a hyperlink that links to another page or resource.
<img> Embeds an image in the document.
<ul> Defines an unordered (bulleted) list.
<ol> Defines an ordered (numbered) list.
<li> Defines a list item, typically used inside <ul> or <ol>.
<table> Defines a table.
<form> Defines an HTML form for user input.
<input> Defines an input field, allowing the user to enter data.
<span> Defines a section of inline text, useful for styling parts of a text.
<strong> Defines important text, typically displayed in bold.
<em> Defines emphasized text, typically displayed in italics.
<br> Inserts a line break (new line) in the text.
<link> Defines the relationship between a document and an external resource (commonly used to link stylesheets).
<meta> Defines metadata about the HTML document, such as author, charset, and description.
<header> Defines the header section of a webpage or a section.
<footer> Defines the footer section of a webpage or a section.
<nav> Defines navigation links in a webpage.
